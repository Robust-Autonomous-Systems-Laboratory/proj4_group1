# proj4_group1
A repository containing Anders Smitterberg and Reid Beckes' implementation of Project 4.

## Usage
1. Create a ros workspace directory
2. Clone this repository into that directory
3. Rename the repository to `src`  
```
mv proj4_group1/ src/
```
4. Initiate git submodules  
```
git submodule init
```
5. Get turtlebot pakage submodules  
```
git submodule update
```
